# E.C.H.O. — Uitleg voor spelers

> Post dit bericht in de groep voordat je de welkomstberichten verstuurt.

```

E.C.H.O. — Welkom, reizigers
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

We gaan samen een zintuiglijk verhaal beleven.
Elk van jullie beleeft het alleen — maar jullie delen
dezelfde wereld. Aan het eind komt alles samen.

Er is geen goed of fout. Geen winnen of verliezen.
Alleen jouw ervaring.

┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈
HOE HET WERKT
┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈

1. Ik stuur je eerst een korte vraag via DM —
   vertel me hoe je genoemd wilt worden, je
   voornaamwoord (hij/zij/hen) en je leeftijd.

2. Daarna krijg je een welkomstbericht via DM.
   Lees het — dit is het begin van jouw verhaal.

3. Na elk hoofdstuk vraag ik wat je voelde.
   Stuur me je antwoord via DM.
   Alles mag — een gevoel, een beeld, een kleur.
   Geen inspiratie? Stuur gewoon "verder".

4. Ik stuur je het volgende hoofdstuk.
   Lees, voel, reageer. Zo gaat het door
   tot het einde.

Je hebt geen AI of app nodig — je leest en
reageert gewoon via DM.

SAMENGEVAT
┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈

  Ik → Jou:   Je krijgt hoofdstukken via DM.
  Jou → Mij:  Je stuurt terug wat je voelde.

Dat is alles. De rest doet het verhaal.
```
